# Keylogger

Simple keylogger listening to keyboard inputs

## Technologies

- Python
- VB Script

## Start keylogger 

```
git clone https://github.com/MatteoBernard/keylogger.git
pip install pynput
```

Change output file path as you want

```
cd .\keylogger\

# Launch with opened terminal
python .\keylogger.py

# Launch in background
.\run-keylogger.vbs
```
